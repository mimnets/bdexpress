## ওভারভিউ

আমরা একটি **মাল্টি-প্ল্যাটফর্ম সিস্টেম** তৈরি করব:
১. **ওয়েবসাইট** (ওয়েব অ্যাপ — PWA সক্ষম)
২. **অ্যান্ড্রয়েড অ্যাপ** (নেটিভ বা ক্রস-প্ল্যাটফর্ম)

- **অটোমেশন-ফার্স্ট** — সাপোর্ট, মার্কেটিং, অপারেশনের জন্য এআই
- **লীন শুরু, দ্রুত স্কেল** — ৬০ দিনে এমভিপি, বাস্তব ব্যবহারের ভিত্তিতে পুনরাবৃত্তি

## ফেজ ১: এমভিপি (সপ্তাহ ১-৮)

### এমভিপি যা করে
- ব্যবহারকারীরা কোট পেতে Taobao/1688 পণ্য লিঙ্ক পেস্ট করতে পারেন
- অ্যাডমিনরা ম্যানুয়ালি অর্ডার পরিচালনা করেন (আধা-স্বয়ংক্রিয়)
- পেমেন্ট bKash/Nagad-এর মাধ্যমে (প্রাথমিকভাবে ম্যানুয়াল ভেরিফিকেশন)
- মৌলিক অর্ডার ট্র্যাকিং
- তথ্য + কোট ফর্মসহ পাবলিক ওয়েবসাইট

### এমভিপি টেক স্ট্যাক

| স্তর | প্রযুক্তি | কারণ |
|------|-----------|------|
| **ফ্রন্টএন্ড (ওয়েব)** | Next.js (React) | দ্রুত, SEO-বান্ধব, PWA প্রস্তুত |
| **মোবাইল** | React Native (Expo) | iOS + Android-এর জন্য একটি কোডবেস |
| **ব্যাকএন্ড API** | Node.js (Express/Nest.js) | মনিরের স্বাচ্ছন্দ্য অঞ্চল |
| **ডেটাবেস** | PostgreSQL | নির্ভরযোগ্য, স্কেলেবল, ORM-এর সাথে ভালো |
| **ORM** | Prisma / Drizzle | টাইপ-নিরাপদ, দুর্দান্ত DX |
| **অথ** | NextAuth.js / Clerk | দ্রুত সেটআপ, সোশ্যাল লগইন |
| **পেমেন্ট** | SSLCommerz + bKash API | স্ট্যান্ডার্ড বিডি পেমেন্ট গেটওয়ে |
| **ক্লাউড** | Vercel + Supabase বা Railway | দ্রুত ডিপ্লয়মেন্ট |
| **ফাইল স্টোরেজ** | Cloudflare R2 / AWS S3 | পণ্যের ছবি, QC ফটো |

## ফেজ ২: অটোমেশন ও API ইন্টিগ্রেশন (সপ্তাহ ৯-১৬)

### মূল বৈশিষ্ট্য
- **Taobao/1688 API ইন্টিগ্রেশন** — লিঙ্ক থেকে স্বয়ংক্রিয়ভাবে পণ্যের বিবরণ আনুন
- **মূল্য তুলনা ইঞ্জিন** — একই পণ্যের জন্য Taobao বনাম 1688 মূল্য দেখান
- **এআই গ্রাহক সহায়তা চ্যাটবট** — বাংলা + ইংরেজিতে প্রশ্নের উত্তর দিন
- **স্বয়ংক্রিয় কোট সিস্টেম** — লিঙ্ক ইনপুট থেকে রিয়েল-টাইম মূল্য নির্ধারণ
- **ট্র্যাকিং অটো-আপডেট** — রিয়েল-টাইম লজিস্টিকস ট্র্যাকিং
- **এফ-কমার্স পোর্টাল** — ফেসবুক বিক্রেতাদের জন্য ডেডিকেটেড টুল
- **কিউরেটেড ক্যাটালগ সম্প্রসারণ** — ৫০০+ পণ্য

### API ইন্টিগ্রেশন অপশন

#### অপশন এ: অফিসিয়াল API (ফেজ ২-এর জন্য প্রস্তাবিত)
| প্ল্যাটফর্ম | API প্রাপ্যতা | নোট |
|-------------|---------------|------|
| **1688.com** | সীমিত (B2B) | অফিসিয়াল API-র জন্য চীনা ব্যবসা নিবন্ধন প্রয়োজন |
| **Taobao** | Taobao ওপেন প্ল্যাটফর্ম | অ্যাপ কী, চীনা সত্তা প্রয়োজন |
| **Alibaba.com** | Alibaba ওপেন প্ল্যাটফর্ম | সবচেয়ে ভালো ডকুমেন্টেড, আন্তর্জাতিক-বান্ধব |

#### অপশন বি: থার্ড-পার্টি এগ্রিগেটর (দ্রুত, ফেজ ১.৫-এর জন্য)
| সেবা | বৈশিষ্ট্য | খরচ |
|------|-----------|------|
| **Onebound** | Taobao/Tmall/1688/জেডিডি পণ্য ডেটা API | প্রতি-রিকোয়েস্ট পে |
| **SimplyBatch** | পণ্য অনুসন্ধান, বিস্তারিত, মূল্য নির্ধারণ API | সাবস্ক্রিপশন |
| **Diffreight** | লজিস্টিকসসহ চায়না সোর্সিং API | কাস্টম মূল্য |
| **Accio** | 1688 সোর্সিং এজেন্ট মার্কেটপ্লেস | কমিশন-ভিত্তিক |

#### অপশন সি: স্ক্র্যাপিং + হিউম্যান ফলব্যাক (এমভিপির জন্য)
- পণ্য পেজ স্ক্র্যাপ করতে Puppeteer/Playwright ব্যবহার (এমভিপি গতির জন্য)
- অস্পষ্ট ক্ষেত্রে মানব যাচাই
- ফেজ ২-এ API-তে স্থানান্তর

## ফেজ ৩: স্কেল ও অপ্টিমাইজেশন (সপ্তাহ ১৭-২৪)

### বৈশিষ্ট্য
- **ইনভেন্টরি ম্যানেজমেন্ট সিস্টেম** — চায়না গুদামে স্টক ট্র্যাক
- **বিটুবি পাইকারি পোর্টাল** — বাল্ক অর্ডারিং, কন্টেইনার শিপিং
- **সেলার ড্যাশবোর্ড** — এফ-কমার্স বিক্রেতাদের তাদের স্টোর পরিচালনার জন্য
- **অ্যানালিটিক্স ড্যাশবোর্ড** — সেলস ট্রেন্ড, শিপিং পারফরম্যান্স, গ্রাহক অন্তর্দৃষ্টি
- **অ্যাফিলিয়েট/রেফারেল প্রোগ্রাম** — ব্যবহারকারীরা অন্যদের রেফার করে কমিশন পান
- **উন্নত অনুসন্ধান** — ইমেজ-ভিত্তিক অনুসন্ধান, ভয়েস অনুসন্ধান
- **মাল্টি-কারেন্সি** — USD, CNY, BDT প্রদর্শন
- **সাপ্লায়ার রেটিং সিস্টেম** — চীনা সাপ্লায়ারদের রেটিং

### পারফরম্যান্স অপ্টিমাইজেশন
- স্ট্যাটিক অ্যাসেটের জন্য CDN (Cloudflare)
- ইমেজ অপ্টিমাইজেশন (WebP, লেজি লোডিং)
- ডেটাবেস কোয়েরি অপ্টিমাইজেশন
- অ্যাপ পারফরম্যান্স প্রোফাইলিং
- লোড টেস্টিং

## ফেজ ৪: সম্প্রসারণ (মাস ৬+)

- **পার্টনারদের জন্য API** — অন্য বিডি ই-কমার্স সাইটগুলিকে আমাদের সোর্সিং ইন্টিগ্রেট করতে দিন
- **ড্রপশিপিং অটোমেশন** — বিক্রেতা → চীন → বিডি পথে স্বয়ংক্রিয় অর্ডার রাউটিং
- **গুদাম ব্যবস্থাপনা মডিউল** — রিয়েল-টাইম ইনভেন্টরি, বারকোড স্ক্যানিং
- **বিডি গুদাম** — জনপ্রিয় পণ্যের জন্য দ্রুত ডেলিভারির জন্য স্থানীয় গুদাম
- **সাবস্ক্রিপশন প্ল্যান** — ঘন ঘন ক্রেতাদের জন্য মাসিক প্যাকেজ
- **লজিস্টিকস অপ্টিমাইজেশন** — রুট অপ্টিমাইজেশন, ক্যারিয়ার তুলনা

- **দ্বিভাষিক (বাংলা + ইংরেজি)** সহজ টগলসহ
- **সরল, ৩-পদক্ষেপ প্রক্রিয়া** প্রকটভাবে প্রদর্শিত
- **আস্থা সংকেত** সর্বত্র: অফিস ঠিকানা, ট্রেড লাইসেন্স, গ্রাহক পর্যালোচনা
- **স্বচ্ছ মূল্য নির্ধারণ** — চেকআউটের আগে খরচ ভাঙ্গন দেখান
- **দ্রুত লোডিং** — বিডিতে ধীর মোবাইল সংযোগের জন্য অপ্টিমাইজ

### ব্র্যান্ড আইডেন্টিটি (পরে নির্ধারিত)
- ব্র্যান্ড নাম, লোগো, কালার স্কিম
- নামকরণের জন্য টিম ইনপুট প্রয়োজন
