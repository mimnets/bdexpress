## ব্র্যান্ড ও আইডেন্টিটি

১. **প্ল্যাটফর্মের নাম কী হওয়া উচিত?** পরামর্শ প্রয়োজন — এমন কিছু স্মরণীয় যা "বাংলাদেশ" + "চায়না সোর্সিং/এক্সপ্রেস" বোঝায়
২. **ডোমেইন নাম:** আমাদের উপলব্ধতা চেক করতে হবে। পছন্দের TLD কী? (.com, .bd, .global?)
৩. **ভাষা:** ডিফল্ট ইন্টারফেস বাংলা নাকি টগলসহ ইংরেজি হওয়া উচিত? সঠিক ভারসাম্য কী?

## তৌহাব (চায়না অপারেশনস)

৪. **চায়না গুদামের অবস্থান:** কোন শহর? (গুয়াংজু/শেনজেন/ইউউ — প্রতিটির আলাদা সুবিধা আছে)
৫. **গুদামের ধারণক্ষমতা ও বর্তমান সেটআপ:** কত বর্গফুট, বর্তমান স্টাফ, কী কী সরঞ্জাম আছে?
৬. **বর্তমান সরবরাহকারী নেটওয়ার্ক:** আপনার কি নির্দিষ্ট Taobao/1688 সরবরাহকারীদের সাথে সম্পর্ক আছে? কোন ক্যাটাগরিতে?
৭. **শিপিং পার্টনার:** আপনি কি ইতিমধ্যে বিডির জন্য নির্দিষ্ট ফ্রেইট ফরোয়ার্ডার বা শিপিং কোম্পানির সাথে কাজ করেন?
৮. **বিদ্যমান ক্লায়েন্ট বেস:** আপনার কি ইতিমধ্যে ক্লায়েন্ট আছে যাদের জন্য আপনি চীন থেকে পণ্য আনেন? আমরা কি তাদের আমাদের প্রথম ব্যবহারকারী করতে পারি?
৯. **গড় শিপিং খরচ:** ঢাকায় এয়ার ও সি-র জন্য বর্তমান প্রকৃত খরচ প্রতি কেজি কত? (এটি আমাদের মূল্য নির্ধারণে সাহায্য করবে)
১০. **চীনে পেমেন্ট:** আপনি বর্তমানে কীভাবে চীনা সরবরাহকারীদের পেমেন্ট দেন? (Alipay, ব্যাংক ট্রান্সফার, WeChat Pay?)
১১. **কাস্টমস অভিজ্ঞতা:** আপনি কীভাবে বিডি কাস্টমস নথিপত্র তৈরি করেন? আপনার কি কাস্টমস ব্রোকার পার্টনার আছে?
১২. **সাধারণ অর্ডার ভলিউম:** আপনি বর্তমানে মাসিক কত অর্ডার/পণ্য প্রক্রিয়া করেন?

## সাব্বির (স্থানীয় অপারেশনস)

১৩. **কোম্পানি নিবন্ধনের অবস্থা:** আপনি কি নিবন্ধন প্রক্রিয়া শুরু করেছেন? সময়রেখা কী?
১৪. **অফিস স্পেস:** আপনার কি ইতিমধ্যে উত্তরায় জায়গা আছে, নাকি খুঁজতে হবে?
১৫. **bKash/Nagad মার্চেন্ট অ্যাকাউন্ট:** আপনি কি আবেদন করেছেন? প্রক্রিয়া ও সময়রেখা কী?
১৬. **লাস্ট-মাইল ডেলিভারি:** আপনার কি কোনো ডেলিভারি সেবার (Pathao, E-Desh, ইত্যাদি) সাথে যোগাযোগ আছে?
১৭. **গ্রাহক সহায়তার পদ্ধতি:** আপনি কি প্রথমে নিজে সহায়তা করবেন নাকি কাউকে নিয়োগ দেবেন?
১৮. **লক্ষ্য বাজারের সাথে সংযোগ:** আপনার কি কোনো এফ-কমার্স বিক্রেতা, দোকানের মালিক বা Daraz বিক্রেতা চেনেন যারা প্রথম গ্রাহক হতে পারেন?
১৯. **প্রাথমিক মার্কেটিংয়ের বাজেট:** প্রথম ৩ মাসে Facebook/Google বিজ্ঞাপনের জন্য আমরা কত বরাদ্দ করতে পারি?

## আকরাম (লজিস্টিকস ও সাপ্লাই চেইন)

২৮. **বর্তমান লজিস্টিকস নেটওয়ার্ক:** আপনি বর্তমানে কোন ফ্রেইট ফরোয়ার্ডার, শিপিং লাইন বা ক্যারিয়ারের সাথে কাজ করেন বা সম্পর্ক আছে?
২৯. **বিডি কাস্টমস ব্রোকার:** আপনার কি বাংলাদেশি কাস্টমস ব্রোকারদের সাথে সম্পর্ক আছে? আপনি কি তাদের রেট শেয়ার করতে পারেন?
৩০. **শিপিং খরচ সম্পর্কে ধারণা:** গুয়াংজু/শেনজেন থেকে ঢাকায় এয়ার ফ্রেইট (প্রতি কেজি) এবং সি ফ্রেইটের (প্রতি সিবিএম) বাস্তবসম্মত বর্তমান রেট কী?
৩১. **পছন্দের গুদাম সংযোগ:** আপনি কীভাবে তৌহাবের চায়না গুদামের কাজকে আন্তর্জাতিক শিপিংয়ের সাথে যুক্ত করবেন — পিকআপ শিডিউল, নথিপত্র হ্যান্ডঅফ?
৩২. **লাস্ট-মাইল ডেলিভারি:** আপনার কি বিডি ডেলিভারি সেবার (Pathao, E-Desh, Sundarban, Paperfly, RedX) সাথে যোগাযোগ আছে? আপনি কি ভালো রেট আলোচনা করতে পারেন?
৩৩. **লজিস্টিকস টেক:** আপনি কি কোনো লজিস্টিকস ম্যানেজমেন্ট প্ল্যাটফর্ম বা TMS (ট্রান্সপোর্টেশন ম্যানেজমেন্ট সিস্টেম) ব্যবহার করেছেন?
৩৪. **সাপ্লাই চেইনের সমস্যা:** আপনার অভিজ্ঞতা থেকে, এখন চীন→বিডি ইম্পোর্টের সবচেয়ে বড় লজিস্টিকস বাধাগুলো কী? আমরা কীভাবে সেগুলো সমাধান করব?
৩৫. **নিয়ন্ত্রক ও আইনি বিষয়:** বিডি কাস্টমসের নিয়ম পরিবর্তন, নিষিদ্ধ জিনিস বা নথিপত্রের প্রয়োজনীয়তা সম্পর্কে কোনো ধারণা যা আমাদের জন্য প্রস্তুতি নেওয়া উচিত?
৩৬. **ক্ষমতা ও উপলব্ধতা:** আপনি BDXpress-এ কত সময় দিতে পারেন? আপনি কি বর্তমানে অন্য লজিস্টিকস প্রকল্পে কাজ করছেন?

## মনির (প্রযুক্তি)

৩৭. **টেক স্ট্যাক নিশ্চিতকরণ:** আপনি কি প্রস্তাবিত Next.js + React Native-তে স্বাচ্ছন্দ্যবোধ করেন, নাকি অন্য কিছু পছন্দ করেন? (Flutter? T3 স্ট্যাক?)
৩৮. **Claude Code সেটআপ:** আপনার কি Claude Code অ্যাক্সেস সেটআপ করা আছে এবং প্রস্তুত? আপনার ডেভ এনভায়রনমেন্ট কী?
৩৯. **হোস্টিং পছন্দ:** Vercel + Supabase বনাম Railway বনাম আপনার Oracle VM-এ সেলফ-হোস্টেড? প্রতিটির আলাদা খরচ/নিয়ন্ত্রণের ট্রেড-অফ আছে।
৪০. **এমভিপির সময়রেখা:** আমরা কি বাস্তবসম্মতভাবে পার্ট-টাইম কাজ করে ৮ সপ্তাহে এমভিপি তৈরি করতে পারি? ফুল-টাইম?
৪১. **API সংযুক্তিকরণের পদ্ধতি:** এমভিপির জন্য ম্যানুয়াল/স্ক্র্যাপিং দিয়ে শুরু করে ফেজ ২-এ অফিসিয়াল API যুক্ত করব? নাকি দিন ১ থেকেই API সংযুক্তিকরণ চেষ্টা করব?
৪২. **পেমেন্ট গেটওয়ে:** আপনি কি আগে SSLCommerz বা bKash API নিয়ে কাজ করেছেন?
৪৩. **মোবাইল অ্যাপের অগ্রাধিকার:** আমাদের কি প্রথমে শুধু ওয়েব লঞ্চ করে ফেজ ২-এ মোবাইল অ্যাপ যোগ করা উচিত, নাকি লঞ্চেই সব প্ল্যাটফর্ম চান?
৪৪. **অন্যান্য প্রকল্প:** আপনার উপলব্ধতা কেমন? আপনার কি অন্য কোনো প্রতিশ্রুতি আছে যা সময়রেখাকে প্রভাবিত করবে?

## সাধারণ ব্যবসা

৪৫. **পার্টনারশিপ কাঠামো:** আমাদের কীভাবে ইক্যুইটি/মুনাফা ভাগ করা উচিত? অপশন:
    - সমান ২৫/২৫/২৫/২৫ ভাগ (৪ অংশীদার)
    - অবদানের ভিত্তিতে (তৌহাবের গুদাম = মূলধন, সাব্বিরের স্থানীয় অপারেশনস = সময়/দক্ষতা, আকরামের লজিস্টিকস নেটওয়ার্ক = সময়/দক্ষতা, মনিরের টেক = সময়/দক্ষতা)
    - বেতন-ভিত্তিক মডেল (প্রত্যেকে বেতন + মুনাফার % নেয়)
৪৬. **বিনিয়োগ:** আমরা কি সবাই শুরুর খরচে (~৫,০০,০০০ বিডিটি মোট) সমানভাবে অবদান রাখব? নাকি বিভিন্ন পরিমাণে?
৪৭. **সময়:** আমরা কখন লঞ্চ করতে চাই? কোন টার্গেট তারিখ আছে? (ঈদ মৌসুম একটি ভালো লঞ্চ উইন্ডো হতে পারে)
৪৮. **ঝুঁকি নেওয়ার মানসিকতা:** সবাই কতটা ঝুঁকি নিয়ে স্বাচ্ছন্দ্যবোধ করেন? (বুটস্ট্র্যাপড বনাম ছোট বিনিয়োগকারী খোঁজা?)
৪৯. **এক্সিট কৌশল:** স্বল্পমেয়াদী চিন্তা (বিক্রির জন্য তৈরি) নাকি দীর্ঘমেয়াদী (জীবনের ব্যবসা হিসেবে তৈরি)?

- [ ] মাসিক ভলিউমের ধারণক্ষমতার অনুমান

- [ ] ফেসবুক পেজ/গ্রুপ সেটআপ
- [ ] বাজার গবেষণা: বিডিতে এখন কোন পণ্য ট্রেন্ডিং

### আকরাম — প্রস্তুত করুন:
