- বিনিময় হার: ১ CNY ≈ ১৫ BDT (আনুমানিক)
- প্রাথমিক মূলধন: অংশীদারদের দ্বারা স্ব-অর্থায়িত

#### অপশন ৩: থার্ড-পার্টি এগ্রিগেটর (বিকল্প/ফলব্যাক)

| প্রদানকারী | মূল্য নির্ধারণ | কভারেজ | সেরা জন্য |
|-----------|---------------|---------|-----------|
| **OT Commerce** (otcommerce.com) | K0 বেসিক: ~$৩০-৫০/মাস<br>K1 স্ট্যান্ডার্ড: ~$১০০-২০০/মাস<br>K2 লাইট: প্রতি-কল পে | Taobao, 1688, Alibaba, Tmall, JD, AliExpress | দ্রুত বাজারে যাওয়া, মাল্টি-প্ল্যাটফর্ম ডেটা |
| **Onebound** (onebound.net) | ~¥৫,০০০-৫০,০০০/বছর API কল ভলিউম + সোর্স কোড লাইসেন্সিং | Taobao, 1688, Tmall | সম্পূর্ণ টার্নকি এজেন্ট সিস্টেম উপলব্ধ (¥২০কে-২০০কে) |
| **SimplyBatch / Accio** | কাস্টম মূল্য | 1688 সোর্সিং | বিডি-নির্দিষ্ট সোর্সিং এজেন্ট |

**সুপারিশ:** ফেজ ১-এর জন্য **ফ্রি 1688 অফিসিয়াল API** দিয়ে শুরু করুন (ফাঁকের জন্য ম্যানুয়াল ফলব্যাক)। যদি সেটি খুব জটিল হয়, তাহলে **OT Commerce K0 বেসিক ($৩০-৫০/মাস)** মধ্যম পথ হিসেবে ব্যবহার করুন। অর্ডার ভলিউম ফ্রি সীমা অতিক্রম করলেই পেইড প্যাকে আপগ্রেড করুন।

#### অপশন ৪: ইন-হাউস স্ক্র্যাপিং (শুধু এমভিপি ফেজ ১)

| মানব ফলব্যাক | শ্রম খরচ | স্ক্র্যাপার যা হ্যান্ডেল করতে পারে না তার জন্য |

**এমভিপির জন্য, আমি এই হাইব্রিড পদ্ধতি সুপারিশ করছি:**
- যেখানে সম্ভব সেখানে পণ্য অনুসন্ধান/বিস্তারিতের জন্য ফ্রি 1688 API ব্যবহার
- না কভার করা পণ্যের জন্য ব্যাকআপ হিসেবে puppeteer স্ক্র্যাপিং ব্যবহার
- কিউসি/সংশোধনের জন্য ম্যানুয়াল মানব পর্যালোচনা
- ফেজ ২ স্কেলে পৌঁছালে পেইড API প্যাকে স্থানান্তর
