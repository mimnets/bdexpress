## ফেজ ১: এমভিপি লঞ্চ (সপ্তাহ ১-৮)

### সপ্তাহ ১-২: ভিত্তি

**মনির (টেক)**
- [ ] TypeScript সহ Next.js প্রকল্প সেটআপ
- [ ] PostgreSQL ডেটাবেস সেটআপ (Supabase)
- [ ] প্রমাণীকরণ সিস্টেম (Clerk / NextAuth)
- [ ] মৌলিক API কাঠামো
- [ ] ডিজাইন সিস্টেম / UI কম্পোনেন্ট

**তৌহাব (চীন)**
- [ ] গুদামের ঠিকানা, ধারণক্ষমতা, বর্তমান সেটআপ ডকুমেন্ট
- [ ] বর্তমান সরবরাহকারী সম্পর্কের তালিকা
- [ ] গ্রহণ/ক্রয়ের জন্য স্ট্যান্ডার্ড অপারেটিং প্রক্রিয়া তৈরি
- [ ] নমুনা কিউসি প্রক্রিয়া ডকুমেন্টেড

**সাব্বির (বিডি)**
- [ ] কোম্পানি নিবন্ধন প্রক্রিয়া
- [ ] ব্যাংক অ্যাকাউন্ট + bKash মার্চেন্ট
- [ ] উত্তরায় অফিস স্পেস

**আকরাম (লজিস্টিকস)**
- [ ] ফ্রেইট ফরোয়ার্ডার অপশন ম্যাপ ও রেট কোটা নেওয়া
- [ ] কাস্টমস ছাড়করণ প্রক্রিয়া ও ব্রোকার কন্টাক্ট ডকুমেন্ট
- [ ] লাস্ট-মাইল ডেলিভারি পার্টনার গবেষণা ও রেট আলোচনা
- [ ] লজিস্টিকস ওয়ার্কফ্লো নির্ধারণ: চায়না গুদাম → বিডি ডেলিভারি

### সপ্তাহ ৩-৪: মূল বৈশিষ্ট্য

**মনির (টেক)**
- [ ] পণ্য তালিকার পাতা (নির্বাচিত পণ্য)
- [ ] লিংক জমা ফর্ম (Taobao/1688 URL পেস্ট)
- [ ] কোট তৈরি (আপাতত ম্যানুয়াল)
- [ ] অর্ডার ব্যবস্থাপনা সিস্টেম (এমভিপি)
- [ ] অ্যাডমিন ড্যাশবোর্ড (মৌলিক)
- [ ] পেমেন্ট সংযুক্তিকরণ (SSLCommerz + bKash)

**তৌহাব (চীন)**
- [ ] নমুনা কিউসি ফটো প্রক্রিয়া তৈরি
- [ ] সাধারণ শিপিং খরচ ও সময়রেখা ডকুমেন্ট
- [ ] টেস্ট অর্ডার: ১টি পণ্য অর্ডার করুন, বিডিতে শিপ করুন, প্রক্রিয়া ডকুমেন্ট করুন

**সাব্বির (বিডি)**
- [ ] SSLCommerz অ্যাকাউন্ট সেটআপ
- [ ] গ্রাহক সহায়তা WhatsApp নম্বর তৈরি
- [ ] ফেসবুক পেজ ও ইউটিউব চ্যানেল সেটআপ

**আকরাম (লজিস্টিকস)**
- [ ] ফ্রেইট ফরোয়ার্ডার চুক্তি ও রেট চূড়ান্ত
- [ ] কাস্টমস ব্রোকার পার্টনারশিপ নিশ্চিত
- [ ] ২টি লাস্ট-মাইল ডেলিভারি সেবার সাথে পার্টনার
- [ ] লজিস্টিকস এসওপি ডকুমেন্ট তৈরি

### সপ্তাহ ৫-৬: ফ্রন্টএন্ড ও মোবাইল

**মনির (টেক)**
- [ ] ওয়েবসাইট ফ্রন্টএন্ড (হোম, সম্পর্কে, মূল্য, কীভাবে কাজ করে)
- [ ] মোবাইল অ্যাপ (React Native) — মূল স্ক্রিন
- [ ] বাংলা + ইংরেজি ভাষা পরিবর্তন
- [ ] খরচ ক্যালকুলেটর
- [ ] অর্ডার ট্র্যাকিং পাতা

**সব অংশীদার**
- [ ] নির্বাচিত পণ্যের ক্যাটালগ ভরাট (৩০-৫০টি পণ্য)
- [ ] মূল্য নির্ধারণ পাতা তৈরি
- [ ] FAQ কনটেন্ট লেখা (বাংলা + ইংরেজি)
- [ ] ৩টি শিক্ষামূলক ইউটিউব ভিডিও রেকর্ড

### সপ্তাহ ৭-৮: টেস্টিং ও লঞ্চ

## মার্কেটিং ও গ্রোথ কৌশল

### লঞ্চের আগে (সপ্তাহ ১-৬)
- "বাংলাদেশ থেকে চায়না ইম্পোর্ট" নিয়ে ফেসবুক গ্রুপ — কমিউনিটি তৈরি
- ইউটিউব শিক্ষামূলক কনটেন্ট (কীভাবে ইম্পোর্ট করবেন, পণ্য রিভিউ)
- তৌহাবের বিদ্যমান ক্লায়েন্ট নেটওয়ার্ক — ওয়ার্ম ইন্ট্রোডাকশন
- SEO কনটেন্ট (চায়না ইম্পোর্টিং নিয়ে বাংলা আর্টিকেল)

### লঞ্চ (সপ্তাহ ৭-১০)
- ফেসবুক ও ইনস্টাগ্রাম বিজ্ঞাপন (লক্ষ্য: ছোট ব্যবসার মালিক, এফ-কমার্স বিক্রেতা)
- লঞ্চ ডিসকাউন্ট: প্রথম অর্ডারে ফ্রি শিপিং
- রেফারেল প্রোগ্রাম: বন্ধুকে রেফার করুন, দুজনেই বিডিটি ২০০ ছাড় পান
- ফেসবুক লাইভ ইভেন্ট: প্ল্যাটফর্ম নিয়ে প্রশ্নোত্তর

### গ্রোথ (মাস ৩-৬)
- ই-কমার্স নিয়ে বিডি ইউটিউবার/ইনফ্লুয়েন্সারদের সাথে পার্টনারশিপ
- এফ-কমার্স সেলার পার্টনারশিপ
- কলেজ/বিশ্ববিদ্যালয় অ্যাম্বাসেডর প্রোগ্রাম
- WhatsApp ব্রডকাস্ট মার্কেটিং

### স্কেল (মাস ৬+)
- ইচ্ছাকৃত কীওয়ার্ডের জন্য গুগল বিজ্ঞাপন
- বিটুবি ট্রেড শোতে অংশগ্রহণ
- Daraz বিক্রেতাদের সাথে কৌশলগত পার্টনারশিপ
- বিডি ব্যবসা মিডিয়ায় প্রেস রিলিজ
